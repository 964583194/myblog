git安装hexo博客

1.github创建一个远程仓库

2.安装hexo工具

>npm install hexo -g

3.测试是否安装成功

>hexo -v
3.1设置npm的镜像源
>npm install -g cnpm --registry=https://registry.npm.taobao.org 
检查是否设置成功 
>cnpm -v

4.hexo下载blog

在blog目录下输入命令初始化文件夹
>hexo init

5.安装依赖

>npm install  

生成静态文件
>hexo g
开启服务器
>hexo s

6.在网页登录blog

localhost：4000

7.安装hexo-deployer-git

>npm install hexo-deployer-git --save
